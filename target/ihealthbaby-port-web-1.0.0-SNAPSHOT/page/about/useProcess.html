欢迎孕妈妈即刻加入春闱远程胎儿监护体系

下载并安装软件
请前往应用商店搜索“微胎心”，下载并安装软件。需要IOS7.1或更高版本，目前仅支持IPHONE5及以上机型，暂不支持IPAD。
完善个人信息
首次启用请注册软件，并正确您的孕周及相关信息，以便医生咨询
绑定微胎监仪
将微胎监仪探头从插座上拔下，输入微胎心监测仪SN号码，点击“绑定”按钮完成仪器绑定。
启动监测
将微胎监仪探头从卡座上拔下，点击软件底部导航“监测”进入监测页面，点击监测页面中的“连接”按钮，软件会搜索连接设备。
抹耦合剂
设备完成连接后，涂上耦合剂，放于腹部寻找最佳胎心监测位置，以声音清楚为准。
固定探头
用胎监带固定探头
开始监测
可在“设置”中选择“自动监测”和“自动结束”功能；也可手动点击 立即开始 来正式启动监测过程。
保存并上传记录
记录可以保存在本地，也可上传云端，只有云端记录才可以发给医护人员咨询。
咨询医生
如果您有咨询权限，可以向您的负责医护人员咨询。
查看回复
如果医护人员已经回复您的咨询，您可点击查看。
