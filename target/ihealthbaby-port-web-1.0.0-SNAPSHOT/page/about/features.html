随时监护胎儿
用于产前胎儿监护，可以随时，精准，连续获取并记录胎儿心率和胎动情况，行专业产前NST（non stress tesst,无应激试验）。NST是全世界妇产科界公认的，使用最为广泛的监测胎儿健康的手段
远程咨询医生
可以根据需求，随时远程提交NST结果给您产检医院的专门医护人或者远程胎儿监护中心值班的医护人员，由他们分析判断。以便在有可疑胎儿受损识别，能为您提供危害胎儿的渐进性的代谢性酸中毒前干预的时机。
及时干预处理
医护人根据可疑胎儿受损程度的识别，将可能指导您居家采取一些干预措施，或者继续监护，或者可能会建议您立即到医院做进一步的检查或者采取临床干预措施。
许多研究已 经表明专业远程胎儿监护：
  1. 可以显著提高NST异常检出率
  2. 可以显著减少早产和新生儿窒息发生
