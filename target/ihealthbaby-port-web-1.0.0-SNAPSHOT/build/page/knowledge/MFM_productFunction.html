与本APP适配的微胎心监测仪(Mini Fetal Monitor ,MFM)，适用于孕中晚期。

1.随时监护胎儿

用于产前胎儿监测，可以连续获取并记录胎儿心率和胎动情况，专业做产前NST(non stress test)。NST是妇产科界公认的、使用最广泛的监测胎儿宫内健康的手段。

2.远程专业咨询

可以根据需要，通过APP提交监测结果给您的产检医院的专门医护人员或者远程胎儿监测中心值班医护人员，由他们分析判断。以便在有可疑受损识别时，能为您提供对危害胎儿的渐进性的代谢性酸中毒的提前干预时机。
